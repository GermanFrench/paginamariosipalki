## Colaboración

Si fuiste agregado como colaborador y no puedes hacer `push`, sigue estos pasos:

1. **Acepta la invitación**: Revisa tu correo electrónico y acepta la invitación de GitHub al repositorio, o visita directamente:
   `https://github.com/GermanFrench/paginamariosipalki/invitations`

2. **Clona el repositorio** (si aún no lo tienes):
   ```bash
   git clone https://github.com/GermanFrench/paginamariosipalki.git
   ```

3. **Configura tu identidad de Git** (si es la primera vez):
   ```bash
   git config --global user.name "Tu Nombre"
   git config --global user.email "tu@email.com"
   ```

4. **Para subir cambios**:
   ```bash
   git add .
   git commit -m "Tu mensaje de commit"
   git push origin main
   ```

Consulta [CONTRIBUTING.md](CONTRIBUTING.md) para más detalles.
